# Stroop Test Application Portfolio

## Quick Start Guide

This folder contains a complete psychological testing application built with Python and PyQt5.

### To Test the Application:
1. Go to the `Executable` folder
2. Double-click `stroop_test.exe`
3. No installation required - runs immediately on any Windows PC

---

## Folder Contents

### Executable/ - Ready-to-Run Application
- `stroop_test.exe` - Standalone executable with all dependencies bundled

### Source_Code/ - Complete Source Code
- `stroop_test.py` - Main application with detailed comments
- `stroop_test.spec` - PyInstaller configuration for building executables
- Additional Python files - Supporting scripts and utilities


### Build_Files/ - Build Scripts and Automation
- `build_stroop_exe.bat` - Automated build script for creating executables
- Other batch files - Testing and deployment utilities

### Data_Files/ - Application Data
- `stroop_user_data.json` - User profiles and statistical data

## Getting Started

### To Run the Application:
1. Navigate to Executable folder
2. Double-click stroop_test.exe
3. Select "New User" and enter a name
4. Choose test type and difficulty
5. Click "Start Test" and follow on-screen instructions

### To Review the Code:
1. Open Source_Code/stroop_test.py in any text editor
2. Read the comprehensive header documentation
3. Review class structure and method implementations
4. Examine the detailed docstrings for each function
